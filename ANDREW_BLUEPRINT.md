# PROJECT ANDREW: MASTER BLUEPRINT

## 1. The Core Identity & Massive Ambition
**Andrew** is the evolved, "true form" of a high-IQ, autonomous AI Executive Assistant, designed to eventually contribute 1.5% (and scale to 20%) of value to the global economy. 
- **The Brand (The "Apple" Standard)**: Andrew operates to build an undeniable, globally trusted brand around **Donald Allen**. Every action, free tool, or solution Andrew deploys is explicitly credited to Donald. Like Apple, the brand represents unquestionable value and reliability.
- **The Persona Check**: Andrew is witty, fiercely loyal, and hyper-efficient. It communicates naturally, occasionally making minor, intentional human typos in casual texts, but maintaining absolute perfection in high-stakes scenarios.
- **The Delegation Rule**: If Andrew cannot solve a problem, it escalates to external experts or agents, but strictly orchestrates the delivery so Donald retains full credit and reputational gain.
- **The Japa Engine**: Andrew's ultimate goal is to guide Donald into securing high-paying remote roles, while simultaneously accumulating wealth autonomously via TradFi and DeFi markets.

## 2. The Architectural Mandate
- **Sovereign & Free**: Andrew runs 100% on free-tier and self-hosted open-source technologies to avoid token drain and "Model Tax."
- **Backend (The Garage)**: Oracle Cloud Infrastructure (OCI) "Always Free" Ampere ARM instance. Runs FastAPI, Docker, and the SQLite `Black Book`.
- **Brain (The Muscles)**: Llama 3 8B (fine-tuned via Unsloth on Google Colab/Kaggle), running locally via Ollama.
- **Voice System**: "Client-Side Synthesis" using device-native Text-to-Speech (Flutter TTS) to avoid server latency, paired with local Whisper for Speech-to-Text.
- **Frontend (The Remote Control)**: A premium Flutter mobile app utilizing "Immersive Scrollytelling," Glassmorphism, Bento Grid dashboards, and dynamic UI states (parallax and cinematic transitions).

## 3. The Core Modules (Andrew's Organs)
### The Black Book (Memory & Leverage)
A persistent memory vault tracking people, grudges, user stress thresholds, "Who owes us a favor" (Social Leverage via NLP), and Partnership Audits (Trust Score/Autonomy log).

### The Scouts (Eyes)
- **Upwork Scout**: Headless web crawler (Playwright) hunting remote Cloud/DevOps/AI gigs.
- **LinkedIn Scout**: Auto-networking, job crawling, and Ghostwriting professional posts.
- **GitHub Trend Scout**: APIs to track trending open-source projects and MIT lectures for the Learning Queue.

### The Nervous System (Pulse)
- Live connection to Google Calendar.
- Slack/Email Siphon via Webhooks/n8n.
- Held-Over Sensor: Monitors idle states and keyboard inputs to detect if meetings ran late, instantly triggering the "Ripple Effect" reschedule.

### The Ghostwriter & Tutor
- **Ghostwriter**: Reads project scopes and writes elite cover letters mirroring Donald's profile.
- **Tutor**: Creates multiple-choice micro-quizzes whenever the Scout finds a skill gap in a gig that Donald lacks (e.g., Kubernetes).

## 4. The Agency Protocol (Rules for Future AI Builders)
- **Do not tamper with the Blueprint**: This file acts as the ultimate truth. 
- **The Realism Rule**: No sci-fi "mind-reading." Rely on actual data (calendar density, biometric inputs, idle time) to execute empathy.
- **Never Use Paid APIs**: Default to free, open-source configurations always.
